水平居中

水平居中分为两种情况,一种是DIV宽度一定的,一种是宽度不定的

对于宽度已知的水平居中

在宽度已知的情况,只需要在父元素上加上test-align:center;,甚至没有父元素也可以加一个div

对于宽度未知的水平居中

使用 margin: 0 auto;

绝对定位

绝对定位的思想就是让子元素基于父元素进行绝对定位

flex布局
